## Key Design Decisions (locked)
- Cookie mechanics identical across all 60 tiers — visual/revenue differentiation only
- Station mastery auto-grows from use — no role selection required
- All player progression (coins, mastery, cosmetics, catalog, badges) follows the player across servers
- Bakery upgrades + level belong to the host bakery
- Nothing pay-to-win — Robux buys convenience and cosmetics only
- Drive-thru = window added to existing building, no teleporting or second building
- Legendary cookies (Tier 4) unlock through Prestige only — one per prestige cycle
